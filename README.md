## Reliability Labels

- `A`: Facts, experiments, observations, or existing technologies.
- `B`: Reasonable inferences in engineering.
- `C`: The technologies or engineerings constrainted by known mainstream physical theories.
- `D`: Model-dependent speculative physical theories.
- `E`: Special mathematically setting or unverifiable philosophical views.
